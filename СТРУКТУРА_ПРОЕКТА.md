# Структура проекта AEngine

## 📁 Корневая директория

### Основные скрипты сборки APK

- **proactive_monitor.sh** ⭐ - Проактивный фоновый мониторинг (рекомендуется)
- **build_apk.sh** - Автоматическая сборка (все в одном)
- **preplace_deps.sh** - Предварительное размещение зависимостей
- **simple_inject.sh** - Инъекция во время сборки (запасной вариант)
- **auto_inject_deps.sh** - Мониторинг с inotify (запасной вариант)

### Диагностические скрипты

- **final_check.sh** - Проверка готовности системы
- **quick_test.sh** - Быстрая диагностика
- **test_buildozer_structure.sh** - Детальная диагностика структуры buildozer

### Документация

- **НАЧНИТЕ_ОТСЮДА.md** - Главная инструкция (начните с этого файла)
- **ПРОАКТИВНЫЙ_МОНИТОРИНГ.md** - Подробное руководство по проактивному мониторингу
- **ANDROID_BUILD_GUIDE.md** - Полное руководство по сборке Android APK
- **README.md** - Основная документация проекта

## 🎯 Рекомендуемый workflow

### Первая сборка

```bash
# 1. Проверка системы
bash final_check.sh

# 2. Загрузка зависимостей (если нужно)
python3 APM/scripts/predownload_deps.py

# 3. Терминал 1: Мониторинг
bash proactive_monitor.sh

# 4. Терминал 2: Сборка
apm build app
```

### Последующие сборки

```bash
# Терминал 1: Мониторинг
bash proactive_monitor.sh

# Терминал 2: Сборка
apm build app
```
